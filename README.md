# Employee Record Manager
Overview

Employee Record Manager is a command-line application developed using Python and Pandas for managing employee data efficiently. The project provides a simple menu-driven interface that allows users to add, view, and remove employee records while storing data permanently in a CSV file.

This project was created to practice and strengthen core Pandas concepts such as DataFrame creation, data manipulation, CSV file handling, row operations, filtering, and basic data management.

# Features
Add new employee records
View all employee records
Remove employee records
Store employee data in CSV format
Menu-driven command-line interface
Persistent data storage
Simple and beginner-friendly implementation

# View Employees

Users can remove an employee record by entering the employee's name. The updated data is then saved back to the CSV file.
